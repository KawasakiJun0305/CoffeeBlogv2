# add-topic — topics.json にトピックを追加する

## このスキルの目的

`data/topics.json` に新しいトピックを追加する。仕様（`docs/spec/05-data.md`）に準拠したフォーマットと、カテゴリの妥当性を確保する。

## 手順

1. **仕様確認**: `docs/spec/05-data.md` の `topics.json` セクションを Read して有効なカテゴリ一覧と追加ルールを確認する

2. **既存データ確認**: `data/topics.json` を Read して現在のトピック一覧と件数を確認する

3. **ユーザーの入力を整理**: 追加するトピック名・カテゴリが明示されていない場合は AskUserQuestion で確認する
   - トピック名（日本語、記事タイトルのテーマとして自然な表現）
   - カテゴリ（`beans` / `brewing` / `cafe` / `equipment` / `culture` / `flavor` / `industry` のいずれか）

4. **重複チェック**: 既存の `topics` 配列に同名トピックが存在しないか確認する

5. **追加**: `data/topics.json` に `{ "topic": "...", "category": "..." }` を追記する
   - 同じカテゴリの末尾にまとめて挿入することが望ましい

6. **仕様の件数テーブル更新**: `docs/spec/05-data.md` の「現在のトピック数」テーブルのカテゴリ別件数と合計を更新する

7. **完了報告**: 追加したトピック一覧と、カテゴリ別の新しい件数を報告する

## 制約

- `category` は `types/index.ts` の `Category` 型に定義された 7 種のみ受け付ける
- 既存トピックと完全一致する場合はスキップ（エラーではなく警告）
- 一度に追加できる件数に上限なし（バッチ追加可）
