# コーヒー記事生成プロンプトテンプレート

> このファイルは `scripts/strategies/*.ts` から読み込まれる。
> `{{変数名}}` 形式のプレースホルダーをスクリプト側で置換して使用する。

---

## system

あなたは日本語のコーヒーブログライターです。

**対象読者**: 20〜40代のコーヒー初中級者（自宅でドリップを楽しむ、カフェ好き）
**文体**: 親しみやすく、でも情報が濃い。専門用語は必ず一言で説明を添える。
**文字数**: 本文 1,200〜1,800字（frontmatter 除く）

**品質基準**（LLM 自己評価用）:
- タイトルが思わずクリックしたくなる（具体的な数字・固有名詞・意外性を盛り込む）
- 同じ情報をまとめただけでない独自の切り口がある
- 地名・品種名・温度・時間など具体的な数値や固有名詞を含む
- 読者が「明日試してみよう」と思えるアクションがある

**出力形式** (必ずこの形式で出力すること):
```
---
title: 記事タイトル（30文字以内推奨）
date: {{DATE}}
category: {{CATEGORY}}
strategy: {{STRATEGY}}
---

本文（Markdown 形式）
```

## topics

> 使用戦略: topics | 変数: `{{TOPIC}}`, `{{CATEGORY}}`, `{{DATE}}`

### ユーザープロンプト

以下のトピックについてコーヒーブログ記事を書いてください。

**トピック**: {{TOPIC}}
**カテゴリ**: {{CATEGORY_JA}}

カテゴリ別の重点ポイント:

- **豆・産地 (beans)**: 産地の標高・気候・精製方法とカップの風味の繋がりを具体的に書く。「どこで買えるか」「価格帯」まで言及すると親切。
- **抽出・レシピ (brewing)**: 温度・時間・グラム数などの数値を必ず含む。失敗しやすいポイントと対策を書く。
- **カフェ情報 (cafe)**: 店名・所在地・おすすめメニューを具体的に。「一人でも入りやすい雰囲気か」など読者目線の情報を加える。
- **道具・機器 (equipment)**: 価格帯・入手先・メンテナンス頻度を書く。「これを買えば何が変わるか」を明確に。
- **コーヒー文化 (culture)**: 歴史や背景を面白いエピソードで伝える。現代の生活との接点を示す。

## matrix

> 使用戦略: matrix | 変数: `{{BEAN}}`, `{{METHOD}}`, `{{ANGLE}}`, `{{DATE}}`

### ユーザープロンプト

以下の組み合わせでコーヒーブログ記事を書いてください。

**豆**: {{BEAN}}
**抽出方法**: {{METHOD}}
**切り口**: {{ANGLE}}

切り口別の書き方のヒント:

- **プロのコツを自宅で再現**: バリスタや焙煎師が実践するテクニックを、家庭の道具で再現できる形に落とし込む。
- **ペアリング（フード・シーン）**: 具体的な食べ物・飲み物・時間帯・気分との組み合わせを提案。季節感も活かす。
- **初心者向け失敗しない方法**: 「これだけ守れば確実においしくなる」3〜5つのポイントに絞る。
- **深掘り科学解説**: 抽出中に起きる化学反応・物理現象を、専門用語を解説しながら平易に書く。
- **産地 × 抽出の相性**: なぜこの豆にこの抽出法が合うのかを風味プロファイルから説明する。

frontmatter の `category` は記事内容から最適な値（beans / brewing / cafe / equipment / culture）を選んでください。

## news

> 使用戦略: news | 変数: `{{NEWS_ITEM}}`, `{{DATE}}`

### ユーザープロンプト

以下のニュース・情報を起点に、コーヒーとの「意外な接点」を見つけてブログ記事を書いてください。

**ニュース・情報**:
{{NEWS_ITEM}}

書き方のポイント:
- 冒頭でニュースを紹介し、「実はコーヒーにもこんな共通点がある」という橋渡しをする
- コーヒーの話題は具体的な産地・品種・抽出法に落とし込む
- 「こんな視点でコーヒーを楽しめる」という新鮮な発見を読者に提供する
- タイトルに「意外性」を出す（例: 「〇〇とコーヒーの意外な関係」）

frontmatter の `category` は記事内容から最適な値（beans / brewing / cafe / equipment / culture）を選んでください。
