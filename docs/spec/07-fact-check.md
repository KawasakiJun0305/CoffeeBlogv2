# 仕様: ファクトチェック

> 関連ファイル: `scripts/fact-check.ts`

---

## 機能概要 — v1 (2026-06-14)

生成された記事 Markdown のコーヒー関連の事実的主張を検証し、明らかな誤りや検証不能な断言を検出する。
パイプライン上の位置: `generate → source-citation → fact-check → format-check → legal-check → post-to-note`

---

## 入力 / 前提条件

- 生成済み Markdown ファイルのパス（`output/YYYY-MM-DD-*.md`）
- `GITHUB_TOKEN` 環境変数（AI 評価で使用）
- 単体実行も可（`--file=<path>` オプション）

---

## リスクレベル定義

| レベル | 意味 | パイプライン挙動 |
|-------|------|----------------|
| `low` | 事実的問題なし | 警告なし・続行 |
| `medium` | 要確認の主張あり | コンソール警告・**続行** |
| `high` | 明確な誤り・虚偽の断言あり | コンソールエラー・**投稿をブロック**（exit(1)） |

---

## 処理フロー

```
1. ルールベース一次評価（正規表現・数値範囲チェック）
   → low  → 処理終了（フラグなし）
   → medium 以上 → ステップ 2 へ

2. AI 二次評価（GPT-4o によるコーヒー事実検証）
   → factRisk / factFlags を確定

3. 結果を frontmatter に書き込み
4. High の場合は exit(1) で後続をブロック
```

---

## ルールベース一次評価（チェック項目）

### [F1] 抽出温度の異常値

**対象**: 全戦略

**検出ロジック**: 本文中の温度表記（数値 + ℃）が極端な値
```
抽出温度として 70℃ 未満 または 100℃ 超の数値が記載されている場合
```

**リスクレベル**: Medium

---

### [F2] 根拠なしの「世界一」「最高」断言

**対象**: 全戦略

**検出パターン:**
```
/(世界一(美味しい|高品質|人気|有名)|世界最高の(コーヒー|豆|品質))/
```

**リスクレベル**: Medium（AI 精査で High に昇格可能）

---

### [F3] 数値の極端な誇張

**対象**: 全戦略

**検出ロジック**:
- カフェイン量: 1杯あたり 1,000mg 超（現実的上限は約 400mg/日）
- 価格: 1杯 10万円超（日本語価格表記で `[0-9,]+万円` かつ 10万以上）

**リスクレベル**: High（AI 精査をスキップして即 High）

---

## AI 二次評価（Medium 以上のみ実行）

**モデル**: GitHub Models / GPT-4o（`baseURL: https://models.inference.ai.azure.com`）

**システムプロンプト骨子:**
```
あなたはコーヒー業界の専門知識を持つファクトチェッカーです。
以下のコーヒーブログ記事について、事実的な主張の正確性を評価してください。

評価観点:
- 産地・品種・農園に関する主張の妥当性
- 抽出方法・温度・時間などの数値の妥当性
- 企業・ブランドに関する事実（設立年、国籍、規模等）
- 健康効果に関する主張（科学的根拠の有無）

出力: JSON { "riskLevel": "low|medium|high", "flags": string[], "summary": string }
```

**タイムアウト**: 30 秒（超過時は一次評価結果をそのまま採用）

---

## 出力 / 事後状態

### frontmatter への書き込み

```yaml
---
factRisk: "medium"
factFlags: ["unverifiable-claim"]
factCheckedAt: "2026-06-14T10:00:00Z"
---
```

### フラグ識別子一覧

| フラグ | 対応するチェック |
|-------|----------------|
| `temp-out-of-range` | F1 抽出温度異常 |
| `unsupported-superlative` | F2 根拠なし「世界一」断言 |
| `extreme-value` | F3 数値の極端な誇張 |
| `unverifiable-claim` | AI 評価: 検証不能な主張 |
| `contradictory-claim` | AI 評価: 事実と矛盾する主張 |

### コンソール出力

```
[fact-check] ✓ low  — output/2026-06-14-foo.md
[fact-check] ⚠ medium [unverifiable-claim] — output/2026-06-14-bar.md
[fact-check] ✗ high  [extreme-value] — 投稿をブロックしました
```

---

## 異常系・エッジケース

| 条件 | 期待される挙動 |
|------|--------------|
| AI 二次評価が API エラー | 一次評価結果を採用してパイプライン続行 |
| AI 二次評価がタイムアウト | 同上 |
| frontmatter が壊れている | チェックをスキップ・警告のみ（ブロックしない） |
| `--skip-fact` フラグ | 全チェックをスキップ（手動実行時のエスケープ用） |

---

## スコープ外

- 外部ファクトデータベース（Wikidata 等）との自動照合
- 画像内テキストのファクトチェック
- 引用元 URL の実際のページ内容照合
